git clone https://github.com/seu-usuario/seu-repositorio , ele vai colocar seu repositorio no vscode.
touch projeto.py , cria os arquivos no repositorio
touch documentacao.txt , cria os arquivos no repositorio
Depois de fazer as alterações nos arquivos acima, clique no botão salvar.
git add . , ele vai pegar os arquivos.
git commit -m "Mensagem" , vai salvar as alterações
git push , irá mandar para o repositorio remoto github.
